# Quantum Sprint Submission: AidAtlas AI
AidAtlas AI is a social-good resource navigator that converts a user situation into a structured needs profile, prioritized support categories, and an actionable follow-up plan.

## What judges can test in under 60 seconds
1. Open app.
2. Click a demo profile.
3. Generate plan.
4. Review tailored recommendations, checklist, urgency tags, and progress tracker.

## Innovation
- Structured AI output directly mapped into a practical case dashboard.
- Graceful AI fallback means the prototype always works live.

## Functionality
- End-to-end intake → analysis → saved dashboard flow.
- Seeded demo cases for instant interaction.
